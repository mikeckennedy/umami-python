## active_users_async()


Retrieves the active users for a specific website.


Usage

``` python
active_users_async(website_id=None)
```


## Parameters


`website_id: Optional[str] = None`  
OPTIONAL: The value of your website_id in Umami. (overrides set_website_id() value).


## Returns


`int`  
The number of active users.
